## Overview

Ranked AI provides a centralized platform where you organize, manage, and collaborate on all your project documentation. You create custom spaces for different projects, add markdown pages, and integrate with your workflows seamlessly. This documentation space serves as your hub for maintaining up-to-date guides, APIs, and notes.

<Callout kind="info">
Ranked AI supports unlimited projects and collaborators, with real-time editing and version history.
</Callout>

## Key Benefits

You gain efficiency in documentation management through these core advantages:

- **Centralized Organization**: Keep all docs in one searchable space.
- **Collaboration Tools**: Invite teams for real-time edits and comments.
- **Version Control**: Track changes with git-like history.
- **Customizable Spaces**: Tailor layouts and permissions per project.

## Quick Start

Get up and running in minutes:

<Steps>
  <Step title="Create Account" icon="user-plus">
    Visit `https://dashboard.rankedai.com` and sign up with your email.
  </Step>

  <Step title="New Space" icon="folder-plus">
    Click "New Space" and name your project, e.g., "MyApp Docs".
  </Step>

  <Step title="Add Pages" icon="file-plus">
    Create your first page with markdown. Use the editor for rich content.
  </Step>

  <Step title="Invite Team" icon="mail">
    Share the space link or add collaborators via email.
  </Step>
</Steps>

For API access, generate a key:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.example.com/v1/spaces', {
    headers: {
      Authorization: `Bearer ${YOUR_API_KEY}`
    }
  });
  const spaces = await response.json();
  console.log(spaces);
  ```

  ```python
  import requests

  headers = {'Authorization': f'Bearer {YOUR_API_KEY}'}
  response = requests.get('https://api.example.com/v1/spaces', headers=headers)
  spaces = response.json()
  print(spaces)
  ```
</CodeGroup>
