## Prerequisites

Before starting, ensure you have:

<Callout kind="info">
- A modern web browser like Chrome, Firefox, or Safari
- An email address for account verification
- Optional: Your `{API_KEY}` from previous integrations
</Callout>

<Callout kind="tip">
Ranked AI works best on desktop for the initial setup.
</Callout>

## Create Your Account and Log In

Follow these steps to get your Ranked AI account up and running.

<Steps>
  <Step title="Visit the Dashboard" icon="monitor">
    Navigate to `https://app.rankedai.com` and click **Sign Up**.
  </Step>
  <Step title="Enter Details" icon="user">
    Provide your email, create a strong password, and verify your account via email.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to the dashboard, enter your credentials, and access the main interface.
  </Step>
</Steps>

## Set Up Your First Documentation Space

Once logged in, create a space to organize your projects.

<Steps>
  <Step title="Create Space" icon="plus">
    Click **New Space** in the sidebar and enter a name like "My Project Docs".
  </Step>
  <Step title="Configure Settings" icon="settings">
    Set visibility to private or public, and add a description for your team.
  </Step>
  <Step title="Save and Enter" icon="check-circle">
    Save the space and click **Enter** to view the editor.
  </Step>
</Steps>

## Import Initial Documents

Import your existing documents using one of these methods.

<Tabs>
  <Tab title="File Upload" icon="upload">
    Drag and drop Markdown, PDF, or TXT files directly into the space editor.
    
    Supported formats include `.mdx`, `.md`, `.pdf` (up to 10MB per file).
  </Tab>
  <Tab title="API Import" icon="api">
    Use the API to programmatically import content.
    
    <CodeGroup tabs="cURL,JavaScript">
    ````bash
    curl -X POST https://api.rankedai.com/v1/spaces/{SPACE_ID}/import \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "url": "https://docs.example.com/my-doc.md",
        "title": "Imported Document"
      }'
    ````
    ````javascript
    const response = await fetch(`https://api.rankedai.com/v1/spaces/${SPACE_ID}/import`, {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        url: 'https://docs.example.com/my-doc.md',
        title: 'Imported Document'
      })
    });
    const data = await response.json();
    console.log(data);
    ````
    </CodeGroup>

    <ParamField path="SPACE_ID" param-type="string" required="true">
      Your documentation space identifier.
    </ParamField>

    <ParamField header="Authorization" param-type="string" required="true">
      Bearer token: `Bearer YOUR_API_KEY`.
    </ParamField>
  </Tab>
</Tabs>
