## Recent Updates

<Update label="2024-10-15" description="v1.3.0" tags={["feature", "bugfix"]}>

## New Features

- Added real-time ranking previews for AI models, allowing you to test rankings instantly.
- Introduced webhook support for ranking events at `https://your-webhook-url.com/webhook`.
- New dashboard analytics for model performance metrics.

## Improvements

- Enhanced API response times by 40% for high-volume ranking requests.
- Updated SDKs with better TypeScript support.

## Bug Fixes

- Fixed intermittent failures in batch ranking endpoints.
- Resolved authentication issues with long-lived tokens.

</Update>

<Update label="2024-09-20" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

</Update>
